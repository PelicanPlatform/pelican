## pelican client-agent start

Start the client agent server

### Synopsis

Start the client agent server as a daemon process. The server will listen
on a Unix domain socket and handle job-based transfer requests.

```
pelican client-agent start [flags]
```

### Options

```
      --database string   Path to the SQLite database file for persistence (default: ~/.pelican/client-agent.db)
      --foreground        Run in foreground instead of daemonizing (default: daemonize)
  -h, --help              help for start
      --max-jobs int      Maximum number of concurrent transfer jobs (default: uses ClientAgent.MaxConcurrentJobs parameter, or 5)
```
