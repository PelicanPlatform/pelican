## pelican-server server database backup create

Create a database backup now

### Synopsis

Create a compressed and encrypted backup of the server database.
The backup is written to the configured backup directory
(Server.DatabaseBackup.Location) and encrypted with all available
issuer keys.

```
pelican-server server database backup create [flags]
```
