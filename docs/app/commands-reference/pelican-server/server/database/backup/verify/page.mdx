## pelican-server server database backup verify

Verify a backup can be decrypted

### Synopsis

Verify that a backup file can be successfully decrypted and
decompressed using the available issuer keys, without actually
restoring any data. If no file is specified, the most recent
backup is verified.

```
pelican-server server database backup verify [backup-file] [flags]
```
