## pelican-server server user scope list

List scopes directly granted to a user

### Synopsis

List the scopes directly granted to a user. This does NOT include
scopes the user inherits via group membership or via server config; for
the full effective set inspect /api/v1.0/me/scopes for that user.

```
pelican-server server user scope list <user-id> [flags]
```
