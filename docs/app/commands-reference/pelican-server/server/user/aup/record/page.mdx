## pelican-server server user aup record

Record AUP acceptance on behalf of a user

### Synopsis

Record acceptance of the AUP for a user. Normally users sign the AUP
themselves through /me/aup; this command is the admin escape hatch (e.g.
for migrating an existing acceptance from another system).

```
pelican-server server user aup record <user-id> [flags]
```

### Options

```
  -h, --help             help for record
      --version string   AUP version string the user is being recorded as accepting (required)
```
