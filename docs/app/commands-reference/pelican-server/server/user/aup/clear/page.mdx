## pelican-server server user aup clear

Clear a user's AUP acceptance, forcing them to re-sign

### Synopsis

Wipe a user's recorded AUP acceptance. The user will be redirected
through the AUP-signing flow on their next AUP-gated request. Distinct
from rotating the active AUP itself, which forces every user on the
server to re-accept.

```
pelican-server server user aup clear <user-id> [flags]
```
