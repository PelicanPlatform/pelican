## pelican-server server user set-status

Update user status (active|inactive)

### Synopsis

Set a user's account status to 'active' or 'inactive'.
Display name is a separate, human-only field; use 'pelican server user update'
to change it (or have the user edit it themselves under their profile).

```
pelican-server server user set-status <user-id> [flags]
```

### Options

      --status string   Status: 'active' or 'inactive'
