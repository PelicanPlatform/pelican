## pelican-server server group update

Update a group's name, display name, or description

```
pelican-server server group update <group-id> [flags]
```

### Options

```
      --description string    New group description
      --display-name string   New human-readable display name
  -h, --help                  help for update
      --name string           New machine-readable group name (system admin only)
```
