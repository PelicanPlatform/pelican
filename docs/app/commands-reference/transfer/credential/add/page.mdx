## pelican transfer credential add

Add a new credential to the transfer server

```
pelican transfer credential add [flags]
```

### Options

```
      --access-token-file string   Path to a file containing the access token (reads from stdin if not provided)
  -h, --help                       help for add
      --issuer string              Token issuer URL
      --name string                Name for the credential (required)
```
