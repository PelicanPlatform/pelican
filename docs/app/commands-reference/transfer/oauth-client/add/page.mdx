## pelican transfer oauth-client add

Register a new OAuth2 client on the transfer server

```
pelican transfer oauth-client add [flags]
```

### Options

```
      --client-id string            OAuth2 client ID (required)
      --client-secret-file string   Path to a file containing the OAuth2 client secret (required)
  -h, --help                        help for add
      --issuer-url string           Issuer URL for the OAuth2 provider (required)
      --name string                 Name for the OAuth2 client (required)
```
