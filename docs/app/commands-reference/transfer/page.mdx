## pelican transfer

Manage the transfer server and its resources

### Synopsis

Provide commands to manage credentials, OAuth2 clients, and transfer jobs on a Pelican transfer server.

### Options

```
  -h, --help            help for transfer
  -s, --server string   URL of the transfer server (e.g. https://transfer.example.com:8447)
  -t, --token string    Path to a token file for authentication
```
