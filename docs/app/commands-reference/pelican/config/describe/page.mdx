## pelican config describe

Print documentation for the specified config parameter

### Synopsis

The 'describe' command prints detailed documentation for a specified configuration parameter,
including its type, default value, description, related components, and whether it is deprecated or hidden.

```
pelican config describe [parameter] [flags]
```

### Examples

```
# View documentation for the Server.WebPort parameter
pelican config describe server.webPort
```
