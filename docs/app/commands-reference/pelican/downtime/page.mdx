## pelican downtime

Manage server's own downtime periods

### Synopsis

Provide commands to list, create, update, and delete scheduled downtime periods
for Pelican servers (Origins/Caches). These commands interact with the server's
administrative API endpoint.

### Options

```
  -h, --help            help for downtime
  -s, --server string   Web URL of the Pelican server (e.g. https://my-origin.com:8447)
  -t, --token string    Path to the admin token file
```
