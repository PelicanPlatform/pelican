## pelican

Interact with data federations

### Synopsis

The pelican software allows one to build and interact
with data federations, enabling the sharing of objects and collections
across multiple dataset providers.

### Options

```
      --config string       config file (default is $HOME/.config/pelican/pelican.yaml)
  -d, --debug               Enable debug logs
  -f, --federation string   Pelican federation to utilize
  -h, --help                help for pelican
      --json                output results in JSON format
  -L, --log string          Specified log output file
      --version             Print the version and exit
```
