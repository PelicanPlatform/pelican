## pelican origin web-ui reset-password

Reset the admin password for the web UI

```
pelican origin web-ui reset-password [flags]
```

### Options

```
  -h, --help          help for reset-password
      --stdin         Read the password in from stdin.
      --user string   The user whose password should be reset. (default "admin")
```
