## pelican object share

Generate a string for sharing access to a namespace.
Note the sharing is based on prefixes; all object names matching the prefix will be accessible

```
pelican object share {URL} [flags]
```

### Options

```
  -h, --help    help for share
      --write   Allow writes to the target prefix
```
